## 我要做的最小功能是什么？

我要开发一个部署在 Monad Testnet 上的每日签到 DApp。
用户可以通过网页连接 MetaMask 钱包，确认当前网络为 Monad Testnet，然后点击签到按钮，调用智能合约完成一次链上签到。
签到成功后，页面能够显示：
```text
当前钱包地址
今天是否已经签到
累计签到次数
连续签到天数
本次签到交易哈希
```

本周的最小目标不是完成完整平台，而是跑通：
```text
连接钱包
→ 检查网络
→ 调用合约
→ 完成链上签到
→ 展示签到结果
```

## 谁会使用它？
这个 Demo 主要面向：
- Web3 新手用户
- Monad Testnet 学习者
- 需要进行每日链上打卡的 Builder
- 想体验钱包与智能合约交互的用户
- 用户完成的一个动作是什么？

## 用户打开网页后，连接 MetaMask 钱包，然后点击“每日签到”。

MetaMask 会弹出交易确认窗口，用户确认交易后，签到信息会写入 Monad Testnet。

交易成功后，用户可以在页面中看到签到成功提示、连续签到天数、累计签到次数和交易哈希。

## 我需要读哪 1–3 个文档？
1. Monad Developer Docs

主要查看：
- Monad Testnet 网络参数
- Chain ID
- RPC 地址
- 区块浏览器地址
- 合约部署与交互方式
2. MetaMask Docs
主要查看：
- 如何请求连接钱包
- 如何读取当前账户
- 如何监听账户切换
- 如何切换或添加 Monad Testnet

3. ethers.js 或 viem 文档
主要查看：
- 如何创建钱包 Provider
- 如何获取 Signer
- 如何根据 ABI 和合约地址创建合约实例
- 如何读取合约数据
- 如何发送签到交易并等待确认

目前准备先选择 ethers.js，因为我觉得对新手来说资料更多，也更容易理解。
