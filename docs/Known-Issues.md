# Known Issues

## 当前限制

1. 当前公开仓库主要完成 Solidity 智能合约，React 前端仍待开发。
2. 签到日期基于 `block.timestamp / 1 days`，采用 UTC 日期。
3. 连续签到逻辑尚未通过完整的跨日期自动化测试。
4. 当前测试以 Remix 手动测试为主，没有 Hardhat 自动化测试。
5. 当前没有用户可以直接访问的 Web Demo。
6. 当前没有实现排行榜和数据看板。
7. 区块链交易需要用户在钱包中确认，无法像普通网页按钮一样立即完成。
8. Monad Testnet 数据可能因测试网调整而变化。
